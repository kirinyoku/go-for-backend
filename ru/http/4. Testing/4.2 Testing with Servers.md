# Тестирование с использованием серверов

[`httptest.ResponseRecorder`](https://pkg.go.dev/net/http/httptest#ResponseRecorder) из статьи [Тестирование обработчиков](./testing-handlers) проверяет логику функции в изоляции. Однако часть поведения проявляется только при реальном сетевом взаимодействии:

- поведение [`http.Client`](https://pkg.go.dev/net/http#Client) — следование редиректам, работа [`CookieJar`](https://pkg.go.dev/net/http#CookieJar), таймауты;
- TLS-рукопожатие и параметры защищённого соединения;
- настройки [`http.Transport`](https://pkg.go.dev/net/http#Transport) — прокси, повторные соединения, выбор протокола.

Для этих сценариев пакет [`net/http/httptest`](https://pkg.go.dev/net/http/httptest) предоставляет инструменты запуска полноценных HTTP-серверов на случайных портах локального интерфейса.

## Тестовый HTTP-сервер

[`httptest.NewServer`](https://pkg.go.dev/net/http/httptest#NewServer) запускает HTTP-сервер и возвращает [`*httptest.Server`](https://pkg.go.dev/net/http/httptest#Server). По завершении теста сервер необходимо явно остановить методом [`Close`](https://pkg.go.dev/net/http/httptest#Server.Close).

```go
// integration_test.go
package main

import (
	"fmt"
	"io"
	"net/http"
	"net/http/httptest"
	"testing"
)

func TestGreeting(t *testing.T) {
	ts := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		fmt.Fprintln(w, "Hello, client")
	}))
	// defer гарантирует остановку сервера и освобождение порта даже при падении теста.
	defer ts.Close()

	// ts.URL содержит адрес вида "http://127.0.0.1:PORT".
	resp, err := http.Get(ts.URL)
	if err != nil {
		t.Fatal(err)
	}
	defer resp.Body.Close()

	body, err := io.ReadAll(resp.Body)
	if err != nil {
		t.Fatal(err)
	}

	// fmt.Fprintln добавляет символ переноса строки — он входит в ожидаемое тело.
	if got, want := string(body), "Hello, client\n"; got != want {
		t.Errorf("body: got %q, want %q", got, want)
	}
}
```

## Закрытие сервера через t.Cleanup

Базовые правила тестовых helper-функций (`t.Helper`, `testing.TB`, проверки статусов и тела) разобраны в статье [Тестирование обработчиков](./testing-handlers). Для `httptest.Server` добавляется отдельное правило жизненного цикла: если сервер создаётся в helper-функции, закрывать его нужно через `t.Cleanup`, а не через `defer` внутри helper.

```go
func newTestServer(t testing.TB, handler http.Handler) *httptest.Server {
	t.Helper()
	ts := httptest.NewServer(handler)
	t.Cleanup(ts.Close)
	return ts
}
```

`t.Cleanup` выполнится после завершения вызывающего теста, поэтому сервер останется доступным всё время выполнения тестового сценария.

```go
func TestGreeting(t *testing.T) {
	ts := newTestServer(t, http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		fmt.Fprintln(w, "Hello, client")
	}))

	resp, err := http.Get(ts.URL)
	if err != nil {
		t.Fatal(err)
	}
	defer resp.Body.Close()

	body, err := io.ReadAll(resp.Body)
	if err != nil {
		t.Fatal(err)
	}

	if got, want := string(body), "Hello, client\n"; got != want {
		t.Errorf("body: got %q, want %q", got, want)
	}
}
```

## Тестирование TLS-соединений

[`httptest.NewTLSServer`](https://pkg.go.dev/net/http/httptest#NewTLSServer) поднимает HTTPS-сервер с автоматически сгенерированным самоподписанным сертификатом. Стандартный [`http.DefaultClient`](https://pkg.go.dev/net/http#DefaultClient) этому сертификату не доверяет и вернёт ошибку проверки TLS.

Для таких тестов `*httptest.Server` предоставляет метод [`Client()`](https://pkg.go.dev/net/http/httptest#Server.Client), возвращающий [`*http.Client`](https://pkg.go.dev/net/http#Client), заранее настроенный на доверие именно этому сертификату.

```go
// newTLSTestServer запускает тестовый HTTPS-сервер и возвращает вместе с ним
// преднастроенный клиент. Клиент необходимо получать отсюда, а не использовать
// http.DefaultClient — тот отклонит самоподписанный сертификат.
func newTLSTestServer(t testing.TB, handler http.Handler) (*httptest.Server, *http.Client) {
	t.Helper()
	ts := httptest.NewTLSServer(handler)
	t.Cleanup(ts.Close)
	return ts, ts.Client()
}

func TestSecureEndpoint(t *testing.T) {
	ts, client := newTLSTestServer(t, http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		w.WriteHeader(http.StatusOK)
	}))

	resp, err := client.Get(ts.URL)
	if err != nil {
		t.Fatal(err)
	}
	defer resp.Body.Close()

	if resp.StatusCode != http.StatusOK {
		t.Errorf("status: got %d, want %d", resp.StatusCode, http.StatusOK)
	}

	// resp.TLS содержит параметры согласованного TLS-соединения.
	// nil означает, что соединение прошло без шифрования.
	if resp.TLS == nil {
		t.Error("expected TLS connection, got plain HTTP")
	}
}
```

::: warning
`ts.Client()` — тестовый клиент, связанный с конкретным `httptest.Server`: он доверяет сертификату этого сервера, закрывает свои idle-соединения при `ts.Close()` и может перенаправлять запросы к `example.com` на тестовый сервер. Не используйте его как общий production-клиент.
:::

## Настройка сервера до запуска

[`NewUnstartedServer`](https://pkg.go.dev/net/http/httptest#NewUnstartedServer) создаёт сервер, но не запускает его. Это способ изменить конфигурацию до старта: после вызова [`Start()`](https://pkg.go.dev/net/http/httptest#Server.Start) или [`StartTLS()`](https://pkg.go.dev/net/http/httptest#Server.StartTLS) конфигурацию нужно считать зафиксированной — поздние изменения могут не повлиять на работающий сервер и привести к гонкам данных.

Типичный сценарий — ограничение минимальной версии TLS:

```go
func TestTLSVersionEnforcement(t *testing.T) {
	ts := httptest.NewUnstartedServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		w.WriteHeader(http.StatusOK)
	}))

	// ts.TLS устанавливается до StartTLS — после запуска конфигурацию не изменяют.
	ts.TLS = &tls.Config{
		MinVersion: tls.VersionTLS13,
	}
	ts.StartTLS()
	t.Cleanup(ts.Close)

	resp, err := ts.Client().Get(ts.URL)
	if err != nil {
		t.Fatal(err)
	}
	defer resp.Body.Close()

	if resp.TLS == nil {
		t.Fatal("expected TLS connection")
	}
	// resp.TLS.Version содержит числовой идентификатор версии TLS.
	if got, want := resp.TLS.Version, uint16(tls.VersionTLS13); got != want {
		t.Errorf("TLS version: got %#x, want %#x", got, want)
	}
}
```

Для кастомизации HTTP-сервера без TLS — например, установки таймаутов — используется `ts.Start()`:

```go
ts := httptest.NewUnstartedServer(handler)
// ReadTimeout устанавливается до запуска: после Start() конфигурацию не изменяют.
ts.Config.ReadTimeout = 500 * time.Millisecond
ts.Start()
t.Cleanup(ts.Close)
```

## Проверка редиректов и cookie

Тестовый сервер позволяет проверять сценарии, недоступные для `ResponseRecorder`: как клиент следует цепочке редиректов и сохраняет cookie, установленные в ответе.

```go
func TestClientRedirectsAndCookies(t *testing.T) {
	ts := newTestServer(t, http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		switch r.URL.Path {
		case "/old":
			// Постоянный редирект: клиент должен автоматически перейти на /new.
			http.Redirect(w, r, "/new", http.StatusMovedPermanently)
		case "/new":
			http.SetCookie(w, &http.Cookie{Name: "session", Value: "active"})
			w.WriteHeader(http.StatusOK)
		default:
			http.NotFound(w, r)
		}
	}))

	// CookieJar необходим для автоматического сохранения и отправки cookie.
	// http.DefaultClient не содержит CookieJar — cookie будут проигнорированы.
	jar, err := cookiejar.New(nil)
	if err != nil {
		t.Fatal(err)
	}
	client := &http.Client{Jar: jar}

	resp, err := client.Get(ts.URL + "/old")
	if err != nil {
		t.Fatal(err)
	}
	defer resp.Body.Close()

	// resp.Request содержит последний выполненный запрос после всех редиректов.
	if got, want := resp.Request.URL.Path, "/new"; got != want {
		t.Errorf("final path: got %q, want %q", got, want)
	}

	if resp.StatusCode != http.StatusOK {
		t.Errorf("status: got %d, want %d", resp.StatusCode, http.StatusOK)
	}

	cookies := jar.Cookies(resp.Request.URL)
	if len(cookies) == 0 {
		t.Fatal("cookies: got none, want at least one")
	}
	if got, want := cookies[0].Name, "session"; got != want {
		t.Errorf("cookie name: got %q, want %q", got, want)
	}
	if got, want := cookies[0].Value, "active"; got != want {
		t.Errorf("cookie value: got %q, want %q", got, want)
	}
}
```
