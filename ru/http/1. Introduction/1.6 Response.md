# Структура http.Response

[`http.Response`](https://pkg.go.dev/net/http#Response) описывает уже полученный HTTP-ответ: статус, заголовки, тело и связанную с ним служебную информацию. Эта структура чаще встречается в клиентском коде и тестах, а не внутри серверного обработчика.

Главное отличие от [`http.ResponseWriter`](/ru/net-http/intro/responsewriter) состоит в направлении работы:

- `http.ResponseWriter` используется на сервере, чтобы **записать** ответ;
- `http.Response` используется на клиентской стороне или в тестах, чтобы **прочитать** уже сформированный ответ.

## Основные поля

| Поле         | Назначение |
| :----------- | :--------- |
| `Status`     | Строковое представление статуса, например `"200 OK"`. |
| `StatusCode` | Числовой HTTP-статус, например `200`, `404` или `500`. |
| `Header`     | Заголовки ответа. |
| `Body`       | Поток тела ответа. После чтения его нужно закрыть. |
| `Request`    | Запрос, для которого был получен этот ответ. |

::: warning
Клиентский код должен закрывать `resp.Body`. Незакрытое тело ответа может удерживать сетевое соединение и мешать его повторному использованию.
:::

## Клиентский ответ

В клиентском коде `http.Response` представляет результат уже выполненного запроса. Например, клиент отправляет запрос через `http.Client`, получает `*http.Response` и затем читает из него статус, заголовки и тело.

```go
func checkStatus() error {
    resp, err := http.Get("https://example.com")
    if err != nil {
        return err
    }
    defer resp.Body.Close()

    if resp.StatusCode != http.StatusOK {
        return fmt.Errorf("unexpected status: %s", resp.Status)
    }

    return nil
}
```

## Ответ в тестах

В тестах обработчиков `http.Response` можно получить через [`httptest.ResponseRecorder`](https://pkg.go.dev/net/http/httptest#ResponseRecorder). `ResponseRecorder` реализует `http.ResponseWriter`, сохраняет записанный ответ в памяти и через метод [`Result()`](https://pkg.go.dev/net/http/httptest#ResponseRecorder.Result) возвращает `*http.Response`.

```go
func ExampleResponseRecorder() {
    handler := http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        w.WriteHeader(http.StatusCreated)
    })

    req := httptest.NewRequest(http.MethodPost, "/", nil)
    rec := httptest.NewRecorder()

    handler.ServeHTTP(rec, req)

    resp := rec.Result()
    defer resp.Body.Close()

    fmt.Println(resp.StatusCode)
}
```

Практические приемы тестирования обработчиков разбираются в статье [Тестирование обработчиков](/ru/net-http/testing/testing-handlers).
