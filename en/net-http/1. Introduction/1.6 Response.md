# The http.Response Struct

[`http.Response`](https://pkg.go.dev/net/http#Response) describes an HTTP response that has already been received: status, headers, body and related metadata. This struct appears more often in HTTP client code and tests than inside server handlers.

The main difference from [`http.ResponseWriter`](/en/net-http/intro/responsewriter) is the direction of work:

- `http.ResponseWriter` is used on the server to **write** a response;
- `http.Response` is used on the client side or in tests to **read** an already formed response.

## Main Fields

| Field        | Purpose |
| :----------- | :------ |
| `Status`     | Text form of the status, for example `"200 OK"`. |
| `StatusCode` | Numeric HTTP status code, for example `200`, `404` or `500`. |
| `Header`     | Response headers. |
| `Body`       | Response body stream. It must be closed after reading. |
| `Request`    | The request that produced this response. |

::: warning
Client code must close `resp.Body`. An unclosed response body can keep the network connection occupied and prevent it from being reused.
:::

## Client Responses

In client code, `http.Response` represents the result of a completed request. For example, a client sends a request through `http.Client`, receives a `*http.Response` and then reads its status, headers and body.

```go
func checkStatus() error {
    resp, err := http.Get("https://example.com")
    if err != nil {
        return err
    }
    defer resp.Body.Close()

    if resp.StatusCode != http.StatusOK {
        return fmt.Errorf("unexpected status: %s", resp.Status)
    }

    return nil
}
```

## Responses in Tests

Handler tests can get an `http.Response` through [`httptest.ResponseRecorder`](https://pkg.go.dev/net/http/httptest#ResponseRecorder). `ResponseRecorder` implements `http.ResponseWriter`, stores the written response in memory and returns `*http.Response` through its [`Result()`](https://pkg.go.dev/net/http/httptest#ResponseRecorder.Result) method.

```go
func ExampleResponseRecorder() {
    handler := http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        w.WriteHeader(http.StatusCreated)
    })

    req := httptest.NewRequest(http.MethodPost, "/", nil)
    rec := httptest.NewRecorder()

    handler.ServeHTTP(rec, req)

    resp := rec.Result()
    defer resp.Body.Close()

    fmt.Println(resp.StatusCode)
}
```

Practical handler testing patterns are covered in the article [`Testing Handlers`](/en/net-http/testing/testing-handlers).
